# Unemployment Analysis Using Python

## Project Overview

This project focuses on analyzing unemployment trends in India using Python-based data analysis and visualization techniques. The analysis explores unemployment patterns across different states, examines the impact of COVID-19 on employment, and identifies regional and seasonal trends that can support economic and policy-level decision-making.

---

## Objective

The main objectives of this project are:

* Analyze unemployment rate data across Indian states
* Perform data cleaning and preprocessing
* Conduct exploratory data analysis (EDA)
* Visualize unemployment trends using Python
* Study the impact of COVID-19 on unemployment
* Identify regional and seasonal unemployment patterns
* Generate insights for economic and policy analysis

## Key Insights

* The average unemployment rate was approximately 11.79%.
* COVID-19 significantly increased unemployment rates across India.
* Certain states consistently showed higher unemployment levels.
* Urban areas experienced larger employment disruptions during the pandemic.
* Seasonal fluctuations in unemployment trends were observed.
* Employment levels and labour participation rates strongly influenced unemployment patterns.

---

## COVID-19 Impact Analysis

The analysis revealed a noticeable rise in unemployment during the COVID-19 lockdown period. Economic slowdown, industrial shutdowns, and reduced business activities contributed heavily to increased unemployment rates across several regions.
